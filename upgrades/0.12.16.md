# Upgrade Guide — v0.12.16

<!-- bump: patch -->

## What Changed

1. **Narrative notification messages** (0.12.15): Notification messages in Agent Attention are now human-readable with plain-language explanations instead of technical codes.

2. **Fix command handler** (0.12.16): Users can now reply directly to notification messages with suggested actions. The server handles these commands mechanically — no Claude session is spawned.

Available fix commands (only work in the Agent Attention topic):
- `fix auth` — Generate and save an API authentication token
- `fix dashboard` — Generate a dashboard PIN
- `fix shadow` — Remove local shadow installations blocking auto-updates
- `fix output` — Re-run output sanity check and report results
- `fix lifeline` — Provides instructions for restarting the lifeline
- `clean processes` — Kill external Claude processes to free memory
- `restart` — Request a graceful server restart
- `restart sessions` — Find and clean up stuck sessions

Commands are intercepted before session routing, so they're instant and free (no Claude API usage).

## What to Tell Your User

- **Reply to fix**: "When your agent reports an issue, it now suggests an action you can take. Just reply with the suggested command (like 'fix auth' or 'clean processes') and the agent will handle it automatically — no waiting for a session to start."

## Summary of New Capabilities

| Capability | How to Use |
|-----------|-----------|

| Fix commands | Reply to Agent Attention with the suggested command |
